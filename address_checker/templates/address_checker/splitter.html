Split an Address

This is a sample page that shows off some of my python and ajax skills.

It is hooked up to a python address splitter function, which uses regular expressions to break a two-field address into a seven-field one. It's mostly useful when converting an address database from a limited two-line format to a modern, normalized format, where each piece of the address has its own field.

I've used ajax techniques to query the python script without refreshing the page, and then add another set of fields to add more data into.


Since addresses come in a lot of formats, a two-line database can have a lot of weird formats crammed into those lines. This attempts to parse all of that.

This page is just an example - put in two address lines, and it'll spit out a nicely parsed address, and give you the space to try another one. But if you just wanted to split a single address, you could do that yourself. The real beauty of this sort of function in when you have thousands of records to clean.
